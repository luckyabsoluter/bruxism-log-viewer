# bruxism-log-viewer

# Features
1. Download log
2. View log

# How to download log
1. Pairing bluetooth with Bruxism Recorder
2. Run program
3. Click `Connect` button
4. Click `Downlaod` button

# How to view log
1. Run program
2. Click `Show` button

# How to run program
1. Enter `java -jar bruxism_log_viewer.jar` in terminal
 
or

1. Run `run.bat`
